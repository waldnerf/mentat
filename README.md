# Mentat v1.0

## Personal Knowledge Operating Model

Mentat is a Markdown-first system for thinking, building and sharing reusable knowledge.

It helps transform ideas into structured knowledge assets and generate multiple expressions from them.

Mentat is not a content production system.

It is a knowledge operating system.

---

# Core Idea

Knowledge is created once.

Artefacts are generated many times.

The canonical Markdown knowledge asset is the source of truth.

Derived outputs are generated from that source.

Examples:

Canonical knowledge:

- principles
- frameworks
- models
- concepts
- specifications
- decision records

Derived artefacts:

- website pages
- presentations
- diagrams
- LinkedIn posts
- documentation
- software implementations

# Important Principle

Not every thought becomes a knowledge asset.

Some ideas mature into reusable frameworks, models or concepts.

Some ideas intentionally become publications.

However:

Every published idea should originate from a deliberate thinking process.

## For new ideas

Start with:

Capture the thought without forcing structure.

---

## For developing ideas

Move to:

- potential reuse

# Design Philosophy

Mentat optimises for:

- clarity over volume
- systems over isolated outputs
- reuse over reinvention
- knowledge over content
- signal over noise
